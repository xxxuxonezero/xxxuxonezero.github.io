    爬虫所需要的库

    1. request

      • headers:请求头，模仿成浏览器访问页面（可以用于爬取反爬页面）

    2. bs4

      • bs4能够快速方便的提取网页中的指定内容，拿到网页源码，然后使用它的接口，将网页源码生成一个对象，让后通过这个对象的方法来获取数据
      • BeautifulSoup可以让html格式的字符串生成一个对象
      • lxml是一个解析器

      bs4和lxml都是要通过pip安装的
