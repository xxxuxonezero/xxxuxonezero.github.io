【leetcode】Subarray Sum Equals K
发表于 2020-05-15| 更新于 2020-05-15|leetcode
阅读量:|评论数:

题目描述：

Given an array of integers and an integer k, you need to find the total number of continuous subarrays whose sum equals to k.

Example 1:

Input:nums = [1,1,1], k = 2
Output: 2

Constraints:

  • The length of the array is in range [1, 20,000].
  • The range of numbers in the array is [-1000, 1000] and the range of the integer k is [-1e7, 1e7].

我看到题的第一反应就是用前缀和，但是我一想这个数据量，时间复杂度为O(n^2)，肯定还可以优化啊，但是我菜，所以我就暴力过了这个。

前缀和大概大家都是知道的，公式就是pre[i]=pre[i-1]+nums[i]

这种思路是用pre[j]-pre[i-1]==k（j>=i）来实现的，也就是求[i,j]之间的和，而第二种做法，则是把公式变一下，之后就会发现时间复杂度变为O(n)了。

之后去看了一下官方题解，优化的方法就是用前缀和+哈希表。

定义了一个哈希表，key存储的是前缀和，而value则存储该key出现过的次数。

把法一的公式变一下，得到pre[j]-k==pre[i-1]，[0,i-1]的前缀和在哈希表中，只要通过get（pre[j]-k）就可以得到应该加的数。
