利用javacc实现词法分析和语法分析
发表于 2020-05-17| 更新于 2020-06-11|编译原理
阅读量:|评论数:

javacc是一个集扫描器生成器、解析器生成器于一身的compiler compiler。它是LL解析器生成器，比起LR和LRLA解析器生成器，可处理的语法范围相对狭小，不过它可以支持无限长的token超前扫描，在一定程度上弥补了它的缺点。

javacc的安装

  1. 下载javacc-javacc-7.0.6.zip
  2. 解压
  3. 下载javacc-7.0.6.jar，并重命名为javacc.jar，复制到javacc-javacc-7.0.6\target（target需要自己新建）
  4. 把javacc-javacc-7.0.6\scripts的全路径加入系统变量Path中
  5. 在命令行中输入javacc，若出现如下信息则表示环境配置成功
