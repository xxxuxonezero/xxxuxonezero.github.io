1371. Find the Longest Substring Containing Vowels in Even Counts
发表于 2020-05-21| 更新于 2020-05-21|leetcode
阅读量:|评论数:

Given the string s, return the size of the longest substring containing each vowel an even number of times. That is, ‘a’, ‘e’, ‘i’, ‘o’, and ‘u’ must appear an even number of times.

Example 1:

Input: s = “eleetminicoworoep”
Output: 13
Explanation: The longest substring is “leetminicowor” which contains two each of the vowels: e, i and o and zero of the vowels: a and u.

Example 2:

Input: s = “leetcodeisgreat”
Output: 5
Explanation: The longest substring is “leetc” which contains two e’s.

Example 3:

Input: s = “bcbcbc”
Output: 6
Explanation: In this case, the given string “bcbcbc” is the longest because all vowels: a, e, i, o and u appear zero times.

Constraints:

  • 1 <= s.length <= 5 x 10^5
  • s contains only lowercase English letters.

我之前好像从来都没有做到过状态压缩的题，这是算是涨知识了。

这题如果只是用前缀和的话空间复杂度为O(5n)，时间复杂度为O(n^2)。

偶数是可以通过偶数-偶数，奇数-奇数得到的，那么就可以把元音出现的次数，转换成元音是出现了奇数次还是偶数次。

这就可以用二进制表示了，0表示偶数次，1表示奇数次。

每个元音都可能出现偶数次或奇数次，故共有2^5=32种状态，故定义一个长度为32的数组，用于记录第一次出现该状态的位置i，之后若再次出现该状态j，那么可以得到长度为j-i+1。

通过状态压缩可以把时间复杂度将为O(n),空间复杂度将为常数级。
